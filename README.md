# PSpider
Python3下极为简洁的爬虫框架

### utilities module
定义爬虫需要的工具类/工具函数等

### instances module
定义抓取过程中的fetcher/parser/saver类

### concurrent module
定义多线程/多进程爬取策略,并保证数据同步

### others
1. setup.py为安装文件,可将该框架安装到系统环境中
2. test.py为测试文件,可进行简单的功能性测试
3. pylint.conf是代码检查需要的配置文件
4. demos_yundama封装了yundama接口,方便调用
